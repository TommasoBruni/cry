CRY(T_T)
========

A small crypto library

Block ciphers
-------------

- AES
- DES and Triple DES

### Block cipher mode of operation

- ECB
- CBC
- GCM
- CTR

Message authentication code
---------------------------

- CMAC

Hash
----

### Cryptographic hash

- MD5
- SHA-256

### CRC

- CRC16-CCITT
- CRC16-IBM

### Checksums

- LRC
- 8-bit trivial checksum

Utilities
---------

- Base64 encoder/decoder
